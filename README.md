# ECO
Cпочатку потрібно на своєму комп'ютері зробити БД script.sql там створена структура БД 
Відкриваємо термінал в якому деректива це папка де знаходиться проект 
пишемо npm i
пишемо npm start 
переходимо в браузер и водемо localhost:3000
